# 🌟 Tribute Page – Dr. A. P. J. Abdul Kalam

A responsive tribute webpage dedicated to **Dr. A. P. J. Abdul Kalam**, developed as **Task 1** of the **Oasis Infobyte Web Development Internship**.

## 📖 About the Project

This project showcases the life, achievements, and legacy of Dr. A. P. J. Abdul Kalam through a clean and modern web interface built using HTML and CSS.

## ✨ Features

- Responsive Hero Section
- Biography
- Timeline of Major Events
- Achievements Section
- Famous Quotes
- Books
- Image Gallery
- Clean Card-Based UI
- Footer with Credits

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
WebDev-L2-TributePage/
│── index.html
│── style.css
│── images/
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open the `WebDev-L2-TributePage` folder.
3. Open `index.html` in your browser.
